# Security Policy

We take the security of Tabula seriously. If you believe you have found a
vulnerability, please read the guidance below on how to report it.

## Supported Versions

We only support security updates for the latest released version of Tabula.
Please ensure you are running the most up-to-date release before reporting a
vulnerability.

## Reporting a Vulnerability

Please do not report security vulnerabilities via public GitHub issues. Instead,
send your report privately to:

**saatvik333sharma@gmail.com**

Please encrypt sensitive communications. You can use the following PGP public
key for encryption:

```text

```

## Safe Harbor

We want to encourage security research. We declare that security research
conducted in good faith and in compliance with this policy is considered
authorized, and we will not initiate legal action against researchers.
